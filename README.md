# WOMMO - <img alt="Works On My Machine Only" src="https://img.shields.io/badge/Works%20On-My%20Machine%20Only-brightgreen">

这是一个自用的小米大杂烩模块。

---

### 现有功能

#### 系统桌面

- 恢复 Google 图标
- 快捷菜单添加小窗

#### 系统界面

- 恢复附近分享磁贴
- 移除通知设置白名单
- 锁屏显示充电功率
- 启用 Alpha 充电动画

#### 手机管家/平板管家

- 将应用详情中“消除默认操作”改为“默认打开”设置

#### 屏幕录制

- 强制启用原生录音支持
- 修改码率与帧数范围

#### 应用包管理组件

- 允许从非官方渠道安装系统应用
